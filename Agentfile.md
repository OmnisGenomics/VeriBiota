# Agentfile

## Commands
- build: `lake build`
- test: `make validate-fast`
- test_python: `python3 -m pytest tests/python`
- tasks: `npm run check:tasks`
- profiles: `npm run check:profiles`
- bench_sw: `./bench_sw`

## Notes
Follow task cards in `/tasks`. Update `.agent/state.json` when switching tasks.
